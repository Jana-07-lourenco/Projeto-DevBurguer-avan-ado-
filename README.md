🍔 Dev Burguer - Cardápio Digital & Sistema de Login

Aplicação web moderna e responsiva desenvolvida para simular a experiência completa de um cardápio digital de hamburgueria, incluindo uma interface de autenticação de usuários.

 🚀 Tecnologias Utilizadas
* HTML5* (Estruturação semântica)
* CSS3* (Estilização avançada, Dark Mode e Layout Responsivo)
* JavaScript (Vanilla)* (Lógica do carrinho e interações)

 📌 Diferenciais do Projeto
* Arquitetura Limpa:* Código organizado com total separação de responsabilidades (pastas dedicadas para HTML, CSS e JS).
* Design Profissional:* Interface moderna em modo escuro com alta fidelidade visual.
* Responsividade:* Adaptável para diferentes tamanhos de tela (Desktops e Dispositivos Móveis).
